Open Coworking Directory

Objective

Build an open directory to replace the directory of spaces on the Coworking Wiki.

Technical Overview

Coworking spaces will publish meta-data points about their space on the main page of their website and register their site with the directory. A spider script will be run at least daily to collect the meta-data from each site and construct static files which are accessible globally (github). Javascript widgets can then convert the meta-data into maps and lists that can be easily embedded into other websites or processes.

Meta-data points will only point to publicly accessible data published elsewhere. For example pricing-url vs pricing-info.
